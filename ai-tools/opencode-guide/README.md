## 这篇讲什么

OpenCode 是一款全能型开源 AI 编程工具，支持多模型、多模式、多智能体协作，覆盖从日常代码补全到复杂代码库改造的全方位场景。本指南从核心概念入门，逐步深入 AI 协作心法、多模型混搭、OMO 多智能体架构，最终延伸至企业级安全部署与 MCP 协议生态。

## 三句话速览

**OpenCode 是什么：** 一款开源 AI 编程工具，一个命令即可在终端召唤 AI 来写代码、改 bug、做重构。

**和 GitHub Copilot 的区别：** Copilot 是助手（你主导），OpenCode 是协作者（AI 在你的环境和流程里工作）。

**OMO 是什么：** Observer‑Model‑Operator，多个专业 AI 智能体分工协作，一个负责规划、一个负责执行、一个负责验收。
