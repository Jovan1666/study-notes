# 五、关于 Dify：不写代码也能搭 RAG

> 如果你不想写代码，Dify 提供了可视化的方式搭建 RAG 工作流。

---

## Dify 是什么

Dify 是一个开源的 LLM 应用开发平台，支持通过拖拽配置来构建 AI 应用，包括 RAG 知识库问答系统。

对于不想手写 Python 代码的人来说，Dify 把 RAG 的各个环节都封装好了：

- 文档上传与解析
- 自动分片与向量化
- 知识库管理
- 可视化工作流配置
- 一键部署为 API 或网页应用

---

## 用 Dify 搭 RAG 的基本流程

```
1. 创建知识库
   → 上传文档（PDF、Word、网页等）
   → 配置分片策略
   → 自动完成向量化

2. 创建应用
   → 选择「知识库问答」模板
   → 关联刚才创建的知识库
   → 配置 Prompt 模板

3. 发布
   → 生成分享链接或 API 接口
```

---

## 和手写代码的对比

| 维度 | 手写代码 | Dify 低代码 |
|------|---------|------------|
| 门槛 | 需要 Python 基础 | 拖拽配置，无需编程 |
| 灵活性 | 完全自定义 | 受平台功能限制 |
| 调试 | 需要自己写日志 | 内置可视化调试 |
| 部署 | 自己搭服务器 | 平台托管或 Docker 自部署 |
| 适合 | 工程师、定制需求 | 快速验证、非技术人员 |
