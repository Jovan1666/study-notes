## 这篇讲什么

本文记录了我从实习期间第一次接触 Cursor，到现在回归 VSCode + 插件化模式的完整工具演变历程。涵盖 IDE 类工具、模型厂商原生工具、插件类工具三大类，重点分享每款工具的真实使用感受和最终的取舍原因。工具之间的使用时间有不少重叠，并不是严格按顺序一个用完再换下一个，同一时期可能同时在探索好几款。

## 三句话速览

**工具演变路径：** GUI IDE 追新（Cursor → Windsurf → Kiro → Warp）→ 回归 VSCode + 插件化 + API 自主配置。

**核心转变：** 从追求便捷交互，到追求模型控制权和成本可控；从订阅制，到按量付费走 API。

**当前模式：** VSCode + Claude Code 插件（cc-switch 管理多模型 API）+ Codex 桌面端，两者搭配使用。
