## 这篇讲什么

上一篇讲了 AI 编程工具的选择，这篇把视角往回拉一步，讲大模型本身。这些 agent 或者编程 IDE，可以调用工具，agent 可以看成大模型 + 工具，大模型就是如何调用工具、调用什么工具的大脑。这篇文章基于真实使用体验，从夯到拉给出主观梯队排名，适合同样在选模型的开发者参考。
